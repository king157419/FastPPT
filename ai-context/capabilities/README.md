# Capability Cards

Each file in this directory is a small routing card for one core FastPPT capability.

## Keep each card focused

Include only:

- what the capability is
- when to read it
- the main source-of-truth files
- current status
- important inputs and outputs
- known gaps

## Status vocabulary

- `active`: current and relied on
- `partial`: implemented but not fully productized
- `planned`: intended direction, not current behavior
