# 教师反馈记录（2026-04-22）

来源：高校老师补充问答（同一赛题语境）。

## 1. 旧 PPT 场景下最常见任务

1. 替换过时内容或数据
2. 增加新案例 / 新论文 / 新政策
3. 把图片、图表、视频片段插进现有课件
4.（次高难）多份资料合并成一套新课件

## 2. AI 生成前最希望确认的信息

1. 教学目标
2. 面向学生类型
3. 重点难点

## 3. 初稿后最常见修改动作

1. 只改某一页内容
2. 增加一个案例
3. 删掉太复杂的部分

## 4. 课程级“长期记忆”接受度

- 方案描述：记住某门课常用教材、案例、图表、风格偏好、修订记录
- 教师反馈：愿意使用
- 顾虑：无明确顾虑

## 5. “可用”结果标准

1. 内容正确
2. 结构顺
3. 像老师自己的课
4. 能回到给定资料，不乱编

## 6. 转化为产品约束

1. 默认支持“页级修改”而非整套重生成为主
2. 在生成前增加三项必确认：教学目标 / 学生画像 / 重点难点
3. 在结果评估中增加“可用四标准”作为验收门槛
4. 课程记忆功能可进入 V1 范围（需提供开关与可清除机制）
