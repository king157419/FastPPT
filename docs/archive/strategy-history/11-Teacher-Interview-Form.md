# 教师访谈表（可勾选版）

用途：尽量少打扰老师，用一次短访谈拿到可执行输入。  
建议 3-5 分钟内完成，全部可勾选。

---

## A. 输入起点（单选）

- [ ] A1 已有旧 PPT，需要更新
- [ ] A2 新开课程，没有可复用 PPT
- [ ] A3 有教材/论文/讲义，但没有 PPT

## B. 你最看重的结果（最多选 2 项）

- [ ] B1 结构清晰（按教学逻辑展开）
- [ ] B2 图文并茂（不是纯文字）
- [ ] B3 可直接上课（修改很少）
- [ ] B4 能引用最新案例
- [ ] B5 低幻觉、可追溯来源

## C. 你最不能接受的问题（最多选 2 项）

- [ ] C1 内容空泛、套话多
- [ ] C2 花哨但不相关
- [ ] C3 图片质量差或图文不匹配
- [ ] C4 无法控制风格和密度
- [ ] C5 修改成本高（局部改动困难）

## D. 输出风格偏好（可多选）

- [ ] D1 简洁学术
- [ ] D2 图文并茂
- [ ] D3 商务专业
- [ ] D4 科技感
- [ ] D5 活泼互动（轻）

## E. 内容块偏好（可多选）

- [ ] E1 公式块
- [ ] E2 代码块
- [ ] E3 流程图
- [ ] E4 对比双栏
- [ ] E5 示例题/案例块
- [ ] E6 时间线/图表

## F. 资料来源（可多选）

- [ ] F1 旧 PPT
- [ ] F2 教材 PDF/Word
- [ ] F3 论文或网页资料
- [ ] F4 图片/示意图素材
- [ ] F5 历史课程视频

## G. 修改策略（单选）

- [ ] G1 接受先出初稿再局部修改
- [ ] G2 希望一次成稿，尽量少改

## H. 初稿后最常见修改动作（最多选 3 项）

- [ ] H1 只改某一页内容
- [ ] H2 增加一个案例
- [ ] H3 删掉太复杂的部分
- [ ] H4 替换过时数据或描述
- [ ] H5 增加图片/图表/视频片段

## I. 输出要求（可多选）

- [ ] I1 每页要点不超过 4-6 条
- [ ] I2 重点页必须带图或图表
- [ ] I3 每章至少 1 个案例
- [ ] I4 关键结论附参考来源

## J. 生成前必须确认的信息（可多选）

- [ ] J1 教学目标
- [ ] J2 面向学生类型
- [ ] J3 重点难点

## K. 课程长期记忆意愿（单选）

- [ ] K1 愿意：系统记住教材/案例/图表/风格/修订记录
- [ ] K2 不愿意

## L. “可用”标准（可多选）

- [ ] L1 内容正确
- [ ] L2 结构顺
- [ ] L3 像我自己的课
- [ ] L4 能回到给定资料，不乱编

---

## 最少必答字段（用于系统生成）

1. 输入起点（A）
2. 最看重结果（B）
3. 最不能接受问题（C）
4. 生成前必须确认信息（J）
5. “可用”标准（L）
